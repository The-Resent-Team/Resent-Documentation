# Better Tab List Mod

---

* <Setting>Show ping</Setting>: Toggles rendering of an entity's ping to server in the tab list.
* <Setting>Show ping in numbers</Setting>: Renders ping as a numerical value[^1] instead of an icon texture.
* <Setting>Player Icons</Setting>: Toggles rendering of an entity's skin in the tab list.

[^1]: This is a best guess value and is not guaranteed to report the exact ping at any given moment.
