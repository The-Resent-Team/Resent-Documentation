# Scoreboard Mod

---

* <Setting>Skip Render Numbers</Setting>: Toggle to skip rendering those red numbers on the right side of the scoreboard.
* <Setting>Round Scoreboard</Setting>: Toggle to render scoreboard with rounded corners.
* <Setting>Round Scoreboard Background Color</Setting>: Color of the rounded scoreboard.
* <Setting>Radius</Setting>: Control the radius of rounding.
* <Setting>Vanilla Scoreboard Color 1</Setting>: Used for vanilla rendering. [^1]
* <Setting>Vanilla Scoreboard Color 2</Setting>: Used for vanilla rendering. [^1]

[^1]: These values are only used when <Setting>Round Scoreboard</Setting> is false.
