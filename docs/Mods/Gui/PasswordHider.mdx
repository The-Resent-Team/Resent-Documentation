# Password Hider Mod

:::note

With the right configuration, this can hide any command or message!

:::

---

* <Setting>Commands to hide, separated by semi-colons</Setting>: A comma seperated list of *prefixes*. Read [^1] for more details.
* <SettingSection>Visual Settings</SettingSection>:
  * <Setting>Character to use to hide password</Setting>: Configure a singular *character* that will be used to hide your message.
  * <Setting>Show "hidden" text indicator when hidden</Setting>: Toggles whether an indicator should be rendered if text is being hidden.
  * <Setting>Hidden indicator color</Setting>: Configure the color of the indicator icon, which indicates if text is being hidden.
  * <Setting>Show lock icon when hidden</Setting>: Toggle to render a lock icon when hiding text.

[^1]: If your typed message starts with one of the supplied prefixes, it will be hidden.
